# Pong-py
Creating the classic Pong game in Python using Pygame as i am learning how to
code.


# Requirements, installation and controls
Requirements:
  Python 3.8
  Pygame 1.9.6

Installation:
  Download the repository or clone it
  Run Pong.py

  Controls:
    Player 1:
      w > up
      s > down

    Player 2:
      i > up
      k > down
